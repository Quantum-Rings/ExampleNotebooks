# Open Quantum notebooks

Sample notebooks for [Open Quantum](https://www.openquantum.com). Launch them
free on Binder (no account needed), open them in Colab, or run them locally.

| Notebook | Launch |
| --- | --- |
| Getting started with the Open Quantum SDK | [![Binder](https://mybinder.org/badge_logo.svg)](https://mybinder.org/v2/gh/Quantum-Rings/ExampleNotebooks/HEAD?labpath=OpenQuantum%2Fgetting-started-openquantum.ipynb) |
| PennyLane on Open Quantum | [![Binder](https://mybinder.org/badge_logo.svg)](https://mybinder.org/v2/gh/Quantum-Rings/ExampleNotebooks/HEAD?labpath=OpenQuantum%2Fopenquantum-pennylane-plugin.ipynb) |
| qBraid on Open Quantum | [![Binder](https://mybinder.org/badge_logo.svg)](https://mybinder.org/v2/gh/Quantum-Rings/ExampleNotebooks/HEAD?labpath=OpenQuantum%2Fopenquantum-qbraid-integration.ipynb) |
| Quantum Rings SDK simulation | [![Binder](https://mybinder.org/badge_logo.svg)](https://mybinder.org/v2/gh/Quantum-Rings/ExampleNotebooks/HEAD?labpath=OpenQuantum%2Fgetting-started-quantum-rings.ipynb) |

Each notebook installs its own dependencies in its first cell, so they run
the same on Binder, Colab, and locally with no environment setup.
