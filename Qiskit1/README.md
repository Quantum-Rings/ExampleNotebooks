# Quantum Rings for Qiskit 1.x

Qiskit finance and optimization tutorials adapted to run on the
Quantum Rings SDK with Qiskit 1.x. Launch them on Binder (no account
needed), open them in Colab, or run them locally.

Some notebooks import the `tutorial_magics.py` helper that sits next to
them in this repository. Binder clones the whole repository, so those
just work; on Colab, download the helper from the same folder and upload
it via the Files pane before running.

The shared Binder environment installs the latest Qiskit (2.x). These
notebooks were written for Qiskit 1.x, so if one hits an API change, run
`%pip install "qiskit<2"` in a cell first and restart the kernel.
