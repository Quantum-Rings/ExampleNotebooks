# Quantum Rings for Qiskit 2.x

Quantum Rings SDK examples and adapted Qiskit tutorials for Qiskit 2.x:
basic usage, primitives (Sampler and Estimator), finance, nature, and
optimization. Launch them on Binder (no account needed), open them in
Colab, or run them locally.

Some notebooks import the `tutorial_magics.py` helper that sits next to
them in this repository. Binder clones the whole repository, so those
just work; on Colab, download the helper from the same folder and upload
it via the Files pane before running.
