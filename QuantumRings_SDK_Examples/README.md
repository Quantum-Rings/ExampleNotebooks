# Quantum Rings SDK examples

Examples for the [Quantum Rings](https://www.quantumrings.com) SDK: getting
started, sizing simulations, and computing mirror fidelity. The
`OpenQASM 3 Examples` folder holds sample `.qasm` circuits used by the
notebooks. Launch them on Binder (no account needed), open them in Colab,
or run them locally.
