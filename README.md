# Quantum Rings example notebooks

Jupyter notebooks for [Open Quantum](https://www.openquantum.com) and the
[Quantum Rings](https://www.quantumrings.com) SDK. Every notebook can be
opened in Google Colab or launched free on Binder (no account needed);
see each folder's README for per-notebook launch links.

| Folder | Contents |
| --- | --- |
| [OpenQuantum](OpenQuantum/) | Run circuits on real quantum hardware with the Open Quantum SDK, PennyLane, and qBraid |
| [Qiskit1](Qiskit1/) | Qiskit finance and optimization tutorials on the Quantum Rings SDK (Qiskit 1.x) |
| [Qiskit2](Qiskit2/) | Quantum Rings SDK examples and adapted Qiskit tutorials (Qiskit 2.x) |
| [QuantumRings_SDK_Examples](QuantumRings_SDK_Examples/) | Quantum Rings SDK simulation sizing, mirror fidelity, and OpenQASM 3 samples |

The Binder environment is defined in [binder/requirements.txt](binder/requirements.txt).
